# Data Augmentation for Deep Learning
Automatic Data Augmentation for Deep Learning techniques. 

## Goal
You can generate new data to train neural networks. This is an easy way to prevent the model overfitting.

## Features
- Crop
- Rotate
- Noise
- Flip up
- Change brightness
